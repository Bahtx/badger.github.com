Get Email Forwards

View the user created email forwarding records for a domain. Requires authentication.

  • Command

    GET /domains/:domain_name/email_forwards

  • Sample Request

    GET https://api.badger.com/domains/badger1.com/email_forwards

  • Sample Response

    [ { "id": 15, "domain_id": 2, "username": "email_forwarder", "destination": "JohnDoe@badger1.com", "created_at": "2011-12-19 05:01:23", "updated_at": "2011-12-19 05:01:23" }, { "id": 16, "domain_id": 2, "username": "*", "destination": "*@badger1.com", "created_at": "2011-12-19 05:01:23", "updated_at": "2011-12-19 05:01:23" } ]

Add Email Forward

Add a new email forward record to the specified domain. Requires authentication.

  • Command

    POST /domains/:domain_name/email_forwards

  • Parameters (required values are in bold)

    Name Description
    username The source of the email forward
    destination The destination of the email forward.

  • Sample Request

    POST https://api.badger.com/domains/badger1.com/email_forwards?username=forwarder&destination=*@badger1.com

  • Sample Response

    { "id": 16, "domain_id": 2, "username": "forwarder", "destination": "*@badger1.com", "created_at": "2011-12-19 05:01:23", "updated_at": "2011-12-19 05:01:23" }

Delete Email Forward

Remove an email forward record from the specified domain. Requires authentication.

  • Command

    DELETE /domains/:domain_name/email_forwards/:email_forward_id

  • Sample Request

    DELETE https://api.badger.com/domains/badger1.com/email_forwards/16

  • Sample Response

    { "message": "Email forward destroyed" }
