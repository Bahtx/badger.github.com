Get Linked Accounts

Show all linked accounts for the authenticated user.

  • Command

    GET /linked_accounts

  • Sample Request

    GET https://api.badger.com/linked_accounts

  • Sample Response

    [ { "status": "synced", "site": "godaddy", "id": 1, "domain_count": 0, "login": "fdsa", "last_synced_at": "2012-02-24T18:06:50Z", "created_at": "2012-02-17T00:25:24Z" }, { "status": "linked", "site": "facebook", "id": 2, "domain_count": 0, "login": null, "last_synced_at": null, "created_at": "2012-02-21T20:04:45Z" }, { "status": "linked", "site": "twitter", "id": 3, "domain_count": 0, "login": null, "last_synced_at": null, "created_at": "2012-02-21T20:09:00Z" } ]

Get Linked Account

Show an individual linked account for the authenticated user.

  • Command

    GET /linked_accounts/:id

  • Sample Request

    GET https://api.badger.com/linked_accounts/3

  • Sample Response

    { "status": "linked", "site": "twitter", "id": 3, "domain_count": 0, "login": null, "last_synced_at": null, "created_at": "2012-02-21T20:09:00Z" }

Get Linked Account Remote Info

Return remote information about the linked account, such as username, profile pictures, etc.

  • Command

    GET /linked_accounts/:id/remote_info

  • Sample Request

    GET https://api.badger.com/linked_accounts/3/remote_info

  • Sample Response

    { "status": "linked", "name": "Badger McGee", "profile_image_url": "http://pics.test.com/profile_picture.jpg", "username": "badger" }
