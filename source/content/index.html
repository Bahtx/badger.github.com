--- title: Overview ---

api.badger.com

The domain registrar services provided by Badger.com can be accessed through our RESTful JSON API at api.badger.com. These pages describe in detail each of the methods work. If you have any questions or suggestions, feel free to e-mail us at support@badger.com.

Frontend Source Code

Our frontend website, www.badger.com uses this API directly and can be found on GitHub at https://github.com/badger/frontend. If you'd like to download and run a local copy of our frontend source code, run these commands:

git clone git@github.com:badger/frontend.git
open frontend/index.html

If you'd like to make changes to our frontend, feel free to submit patches through GitHub Pull Requests.

API Docs

These API docs are also available on GitHub at https://github.com/badger/badger.github.com.
