Create Contact

Create a new contact for the authenticated user.

  • Command

    POST /account/contacts

  • Parameters (required values are in bold)

    Name Description
    first_name The first name for the contact.
    last_name The last name for the contact.
    organization The organization that the contact belongs to.
    phone The phone number for the contact.
    fax The fax number for the contact.
    email The email address for the contact.
    address The street address for the contact.
    address2 The extended address for the contact.
    city The city for the contact.
    state The state for the contact.
    zip The five digit postal code for the contact.
    country The two letter alphanumeric country name for the contact.

  • Sample Request

    POST https://api.badger.com/account/contacts?first_name=John&last_name=Smith&phone=1112223456&address=123%20Fake%20Street&city=San%20Francisco&state=CA&zip=12345&country=US

  • Sample Response

    { "message": "Contact created" }

Update Contact

Update one of the authenticated user's contacts.

  • Command

    PUT /account/contacts/:contact_id

  • Parameters (required values are in bold)

    Name Description
    first_name The first name for the contact.
    last_name The last name for the contact.
    organization The organization that the contact belongs to.
    phone The phone number for the contact.
    fax The fax number for the contact.
    email The email address for the contact.
    address The street address for the contact.
    address2 The extended address for the contact.
    city The city for the contact.
    state The state for the contact.
    zip The five digit postal code for the contact.
    country The two letter alphanumeric country name for the contact.

  • Sample Request

    PUT https://api.badger.com/account/contacts/1?first_name=Isaac&last_name=Newton&email=gravity_is_awesome@gmail.com

  • Sample Response

    { "zip": "12345", "country": "US", "fax": "", "city": "San Francisco", "last_name": "Newton", "organization": "", "state": "CA", "id": 3, "label": null, "address2": "", "phone": "1112223456", "email": "gravity_is_awesome@gmail.com", "created_at": "2011-11-11T22:38:28Z", "first_name": "Isaac", "address": "123 Fake Street" }
